## Context

Wire the production build as a hard deploy gate so a failing `npm run build` blocks the deploy instead of shipping a broken page.

_Advances: P3, E-005_

## Acceptance Criteria

- [ ] Vercel build runs `npm run build`; a deliberately-failing build yields a failed deployment that is NOT promoted to the public URL (prior good deploy stays live) — verified by an intentional build-fail run showing the gate held.
