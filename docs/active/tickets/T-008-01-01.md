## Context

Expose a pure lib/ helper that enumerates every legal (rotation, column) placement of the active piece and returns the settled board for each, so the bot can evaluate candidates without advancing bag/score or mutating state.

_Advances: P5, E-008-read-drive-seam_

## Acceptance Criteria

- [ ] A new pure function (e.g. lib/bot-placements.ts enumeratePlacements(board, pieceType)) returns one settled-board result per reachable rotation+column; a unit test asserts, for a constructed board+piece, that each result's occupied cells match a hardDrop of that piece at that rotation/column and that no input board/bag is mutated (suite green).
